# Excel-PowerBI-Mini-Project
FIFA player data analysis and interactive dashboard built using Excel and Power BI.
This project analyzes FIFA player data using Excel and Power BI to create an interactive dashboard. It provides insights into player ratings, potential, age, positions, and clubs.
Tools Used

Microsoft Excel (data cleaning & preparation)

Power BI (data modeling, DAX, dashboards)

 Key Insights

Player Overall Rating & Potential

Top Players and Clubs

Age and Position Distribution

Interactive filters and visuals

 Objective

To showcase data analysis and dashboard creation skills using Excel and Power BI.
